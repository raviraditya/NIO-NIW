# NIO-NIW

MATLAB analysis for investigating tropical cyclone–generated near-inertial wave energetics and vertical energy redistribution in the North Indian Ocean.

This repository accompanies the manuscript:

Singh, R., & Behera, M. R. (2026). *Tropical Cyclone Generated Near-Inertial Energy and Its Vertical Redistribution in the North Indian Ocean.* JGR: Oceans (submitted).

## Repository Contents

### Core Analysis
- **`Main_combined.m`** — Primary diagnostic engine for per-cyclone processing of HYCOM ocean-state outputs. Computes near-inertial velocity decomposition, near-inertial kinetic energy (NIKE), WKB-normalised energetics, vertical group velocity, wind-power input, resonance parameters, and right/left TC-relative partitioning. Set `STORM_ID` to one of: `KYARR`, `AMPHAN`, `FANI`, `TAUKTAE`.

### Post-processing
- **`Post_combined.m`** — Reads the `.mat` and `.xlsx` outputs generated by `Main_combined.m` and computes penetration-depth diagnostics, energy-conversion efficiency, Richardson-number statistics, right/left asymmetry metrics, and Langmuir-turbulence indicators.

### HYCOM–Argo Validation
- **`HYCOM_Argo_validation.m`** — Validates HYCOM upper-ocean temperature, salinity, and mixed-layer depth against quality-controlled Argo profiles. Features:
  - spatiotemporal colocation (50 km / 3 h)
  - QC filtering using Argo quality flags (1 & 2)
  - support for delayed-mode adjusted variables
  - temperature/salinity validation from 5–500 m
  - mixed-layer-depth comparison computed identically for Argo and HYCOM (0.2 °C threshold from a 5 m reference, strictly like-for-like)
  - Murphy Skill Score and other skill metrics

  Produces the validation statistics underlying Text S1, Figure S2, and Table S1 of the Supporting Information.

## Requirements
- MATLAB R2021a or newer
- MATLAB NetCDF support (`ncread`)
- MATLAB Statistics functionality (`corrcoef`, `interp1`)
- [GSW Gibbs SeaWater (TEOS-10) Toolbox](https://www.teos-10.org/software.htm)

## Data Sources
- **HYCOM** — GLBy0.08 reanalysis. Source: https://www.hycom.org/dataserver/gofs-3pt1/reanalysis
- **ERA5** — atmospheric forcing from the Copernicus Climate Data Store (CDS); downloaded as GRIB and converted to NetCDF with no spatial regridding.
- **Argo** — delayed-mode profiles from the Argo Global Data Assembly Centre (GDAC). Source: https://argo.ucsd.edu/
- **Cyclone Tracks** — India Meteorological Department (IMD) cyclone e-Atlas.
